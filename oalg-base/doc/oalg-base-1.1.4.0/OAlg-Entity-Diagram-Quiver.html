Quiver

data Quiver n m Source #

quiver of n points and m arrows.

Property Let Quiver w o be in Quiver n m, then holds: For all 0 <= j < m holds: s j < n and e j < n where n = lengthN w, s j = start (o j) and e j = end (o j).
