Boolean

class Boolean b where Source #

types with a Boolean structure, allowing multivalent logic.

Note Every Enum type which is also Bounded has a natural implementation as false = minBound, true = maxBound, (||) = max, (&&) = min (as there are min and max bounds the operator (||) and (&&) should be implemented with a lazy variant of min and max) and not b = toEnum (fromEnum maxBound - fromEnum t).

Minimal complete definition

true, false, not, ((||), (&&) | or, and)
